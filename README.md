sdruOFDM
========

OFDM implemented in MATLAB for USRP radios

The receiver and transmitter are split into separate branches (rx and tx)

__Complete:__
Modules
 - CRC (3 bits)
 - Custom messages
 - Timeout mechanism
 - Class based control of tx and rx
